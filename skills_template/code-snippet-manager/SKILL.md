# Code Snippet Manager Skill

This skill helps you organize, search, and manage code snippets and templates.

## Capabilities
- Store and retrieve code snippets
- Search snippets by language, tags, or keywords
- Generate code from templates
- Organize snippets by category
- Convert snippets between languages

## Usage Examples

### Snippet Storage
Save frequently used code patterns as reusable snippets with descriptions and tags.

### Search and Retrieval
Quickly find relevant code snippets based on keywords, language, or functionality.

### Template Generation
Generate boilerplate code from templates, customizing for specific use cases.

## Best Practices
- Use descriptive names and tags
- Include usage examples in comments
- Keep snippets focused and single-purpose
- Document parameters and return values
- Update snippets regularly to reflect best practices
