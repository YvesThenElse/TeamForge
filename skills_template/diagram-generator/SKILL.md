# Diagram Generator Skill

This skill helps you create diagrams and visualizations using Mermaid syntax.

## Capabilities
- Generate flowcharts from process descriptions
- Create sequence diagrams for interactions
- Build class diagrams for software architecture
- Design entity-relationship diagrams for databases
- Produce Gantt charts for project timelines

## Usage Examples

### Flowcharts
Convert process descriptions into clear flowcharts showing decision points and process flow.

### Sequence Diagrams
Create sequence diagrams to visualize API calls, user interactions, or system communications.

### Architecture Diagrams
Generate class or component diagrams to document software architecture.

## Best Practices
- Use clear, descriptive node labels
- Keep diagrams simple and focused
- Use consistent styling and colors
- Add legends when necessary
- Organize elements logically (top-to-bottom, left-to-right)
